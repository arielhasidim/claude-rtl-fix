# Claude RTL Fix

Safari Web Extension that automatically applies right-to-left (RTL) direction
to Hebrew and Arabic responses on [claude.ai](https://claude.ai).

## Quick start

This project was generated by `setup_claude_rtl.sh`.

1. Open the `.xcodeproj` in Xcode
2. Set your Team in Signing & Capabilities (both targets)
3. Change bundle ID from `com.yourname.ClaudeRTL` to your own
4. `⌘R` → enable the extension in Safari Settings → Extensions

## Updating selectors

If claude.ai changes its DOM structure, edit `SELECTORS` in `content.js`.
Run in Safari DevTools to find the right element:
```js
document.querySelectorAll('[data-testid="assistant-message"]')
```

## App Store

Archive → Distribute → App Store Connect.
Replace placeholder icons before submission.
